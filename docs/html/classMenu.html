Detailed Description

Menu class to initialize desktop hierarchy, distinguish and simplify the logical connection between different menus of the program.

Member Function Documentation

◆ complexity_menu()

void Menu::complexity_menu ( )

Menu to choose the initial complexity of the singleplayer game. There are 3 choises available:

  • Mechanics (level 1),
  • STIC (level 3),
  • Applied Maths (level 5). It is also possible to return to main menu.
